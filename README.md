# Smart-Contact-Manager

It is smart contact manager application.
In this application we have build it with Java,MySQL,SpringBoot,Spring-security and Html,CSS,JS & Thymleaf 
it has  registration page where new user caan get register and by providing their required information.
After that they can sign in and add contacts 
it has features of uploading contact picture,pet name,description of a contact.
Each user's password is encrypted with spring security of even the developer cannot see what password is stored.
it has many view pages like home page,about,login and registration page.
it is very user friendly.
